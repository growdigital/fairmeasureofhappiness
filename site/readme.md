#Readme
Source code for [fairmeasureofhappines.com][1]. Built using Wintersmith and Handlebars. For more details see my [blog post][2].

##Running the site
```bash
npm install
wintersmith preview
sass --watch stylesheets:contents/css
grunt watch
```
